
Project Name:
             RBAC Authentication System

This Project is a full-stack Role-Based Access Control (RBAC) authentication
system build using Spring Boot for the backend and React for the backend.

This application allows to users to register, login and accress the feature based on role.
Secure authentication using JWT tokens and Spring Security.

Project Features:
  User Registration
  User Login with JWT Authentication
  Role-based Access Control
  Admin Dashboard
  User DashBoard
  Secure REST APIs
  Password Encryption using BCrypt
  Admin and User Dashboard.
